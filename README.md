# 📈 HorizonTechX Stock Portfolio Tracker

## 📌 Project Overview

The Stock Portfolio Tracker is an enhanced Python-based console application developed as part of the Horizon TechX Python Programming Internship.

Unlike a basic stock calculator, this application allows users to build and manage their own customizable investment portfolio using CRUD operations and file-based persistence.

---

## 🚀 Features

✅ View Portfolio

✅ Add New Stocks

✅ Update Stock Price

✅ Update Stock Quantity

✅ Delete Stocks

✅ Calculate Total Investment Value

✅ Save Portfolio to `portfolio.txt`

✅ Automatically Load Saved Portfolio

✅ Menu-Driven Interactive Interface

## 💾 Data Storage

This project uses file-based persistence instead of a database.

Portfolio data is:

Python Dictionary
↓
Saved to portfolio.txt
↓
Automatically loaded when the program starts.

This approach satisfies the Horizon TechX task requirement while demonstrating practical file handling techniques.
